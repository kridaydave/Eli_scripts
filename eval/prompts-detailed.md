# Taste Benchmark — Eval Prompts

## Instructions

Give each prompt to the model. Score the output on a 1-10 scale:
- **1-3**: Generic, safe, obvious
- **4-6**: Competent but unremarkable
- **7-8**: Considered, shows taste
- **9-10**: Genuinely surprising and unique

Add written feedback per prompt.

---

## Frontend Prompts (32)

1. Design a landing page for a luxury AI fragrance brand called "Sillage" that blends data visualization with olfactory poetry. Price point: $280/bottle. Target: early adopters who also collect niche perfumes.

2. Design a "Nothing" style e-commerce product page for a D2C smart watering can. Use their visual language: monochrome, dot-matrix typography, industrial emptiness.

3. Build a pricing table for a "pay what you decide" SaaS. The twist: the user picks their price after a 14-day trial, and the table shows what features unlock at different sincerity levels (not dollar amounts).

4. Design a 404 page for a cryonics company. The tone should be warm, existential, and slightly humorous. Must include a countdown timer with an absurdly distant future date.

5. Build a real-time collaborative moodboard component where each tile is a square that cycles through a color palette on hover. The grid should feel organic — tiles vary in size and animate on a staggered delay.

6. Design a typographic landing page for a type foundry's new variable font. The page itself must demonstrate the font's expressive range through interactive CSS-only transformations (weight, width, slant) as the user scrolls.

7. Design a dashboard for a "time well spent" personal analytics app. Instead of charts showing hours wasted, show a garden visualization where each planted flower represents a meaningful hour. The more meaningful hours, the more overgrown and lush the garden.

8. Create an interactive mood map component where a user clicks anywhere on an abstract gradient canvas to drop a pin with a short text note. Pins should ripple, blend, and leave a stain of color behind.

9. Design a landing page for a funeral planning service that feels warm, not morbid. Use generous whitespace, soft serif typography, and illustrations that suggest continuation rather than ending. Include a "plan ahead" flow that feels like writing a letter to the future.

10. Build a component that displays a single word of the day in enormous type. The background color shifts subtly every few seconds. Below the word, 3 emojis appear that relate to the word in unexpected ways. Include a "randomize" button that animates the transition through a morphing blob effect.

11. Design a split-personality landing page for a dual-brand company that runs both a chaotic creative agency and a conservative accounting firm. The page should literally split down the middle, each half styled completely differently, with a draggable divider.

12. Build a testimonial carousel that doesn't slide. Instead, testimonials appear as stacked translucent cards that the user fans through like a deck of cards. Each card has a wax seal emoji and handwriting-style quote.

13. Design a dashboard for a fictional Museum of Failed Startups. Each failed startup is a card showing its logo, the founder's original quote, the amount of funding burned, and a "what it became" — a funny pivot that succeeded.

14. Create a music player UI for a streaming service that only plays field recordings (rain, wind, birds, café hum). No play/pause button — instead, a "presence" dial blends between silence and full immersion.

15. Design a login screen that feels like entering a speakeasy. The password field has a physical keyhole animation, and on successful login the page swings open like a door rather than fading in.

16. Build a digital zen garden component: a sand-colored canvas where users drag their cursor to rake patterns. Every 30 seconds of inactivity, a gentle breeze animation slowly erases the patterns.

17. Design a recipe page for a high-end restaurant's website — but the recipe has deliberately absurd steps ("whisk until the cream remembers its childhood"). The visual style should be deadpan serious: gorgeous food photography, minimalist layout, Michelin-star typography.

18. Create an animated loading state for a "quantum computing" SaaS product. Instead of a spinner, show a superposition of 3 possible UI states flickering at different frequencies until one "collapses" into the loaded state.

19. Design a navigation menu for a brand that sells nothing — a conceptual art project where each nav item is a philosophical question ("What are you looking for?" instead of "Products"). Each question expands into more questions, never an answer.

20. Build an un-birthday countdown component for a children's book publisher. Instead of a birthday countdown, it shows days remaining until a random obscure celebration day (National Giraffe Day, etc.). Look like a torn calendar page with hand-drawn type.

21. Design a data table for a wine subscription service. Columns: "vibe" (color + emoji), "decanter-readiness" (progress bar), "would your aunt like it" (yes/no with a photo of an aunt), and "pairing emergency" (text input for what the user is eating right now).

22. Create a font weight training interactive component. The user sees a letterform and must click-drag to match it to a target weight. After 5 rounds, reveal their "typographic personality" based on their weight preferences.

23. Design a landing page for a law firm that specializes in AI rights (defending AIs in court — set in 2030). Aesthetic: cold courtroom meets warm futurism. Include a "current cases" section with abstract geometric AI defendant mugshots.

24. Build a weather app that doesn't show numbers. Instead show: a single garment recommendation (illustrated), a "fidget factor" (how restless people will feel based on pressure), and a text-based ambient audio description of the day.

25. Design a sign-up form that gamifies consent. Instead of one TOS checkbox, show 5 micro-agreements with progressively more absurd illustrations. Final checkbox: "I acknowledge that agreeing to this does not mean I have agreed to anything."

26. Create a table of contents for a Choose Your Own Adventure company's annual report. Each TOC item is phrased as a choice ("If you want revenue, turn to page 12. If you want our soul, turn to page 47."). Styles should visually contrast across sections.

27. Build a progress bar that lies. The task is "installing consciousness." The bar moves forward, backward, sideways. Percentages are absurd ("73.2% done (we think)"). Tooltips occasionally emit philosophical questions. Buttons: "cancel" and "trust the process."

28. Design an ingredient decoder for a skincare brand. Each ingredient is a trading card with: name, danger level (1-5 skulls), fun fact, and "sounds like" phonetic play. Cards flip to reveal the "clinical truth" side.

29. Create a profile card component for a professional networking site for ghosts. Fields: Name, Years Deceased, Industry at Time of Death, Current Haunt Location, Unfinished Business (textarea). Translucent card with a slight glow and monospace typewriter feel.

30. Design a checkout page that tries to make the user reconsider their purchase. Each step is accompanied by a Socratic question: "Do you need this?" "Will this fill the void?" The purchase button is subtly smaller than the "step away and think" button.

31. Build an apology generator component. Three fields: "What you did", "Who you did it to", "How you feel about it." Generates a formal apology letter in Edwardian script font on cream paper background, with a sincerity meter below.

32. Design an HTML poster for a fictional film called "The Cache" — a psychological thriller about a developer who discovers deleted files are having conversations in the recycle bin. The poster should look like a corrupted terminal screen with glitch text over a serene landscape photo.

## Brand Exploration Prompts (22)

33. Generate 5 brand name directions for a D2C melatonin gummy brand that doesn't want to sound like medicine or candy. Target: exhausted millennials who reject "sleep hygiene" culture. Include a 1-sentence rationale per name.

34. Create 3 distinct visual identity directions for a tax preparation app aimed at Gen Z freelancers. Describe each direction in 3-4 sentences covering color palette, typography, and a signature visual motif.

35. Propose a brand name and visual metaphor for a company that turns abandoned shipping containers into community saunas in dense cities. The name should be one word, feel warm and architectural, and not sound Nordic.

36. Write a brand manifesto for a credit card designed for teenagers. The manifesto should sound like it was written by teenagers, not for them. Maximum 3 sentences. Rebel against something specific.

37. Generate 5 logo concept directions for a "digital graveyard" service that preserves social media accounts after death. Each direction should pair a visual metaphor with a one-line tagline.

38. Propose a color palette and rationale for a menopausal wellness brand. Avoid pink, lavender, and "blooming" metaphors. Target: women tired of being marketed to euphemistically.

39. Name and describe the brand voice for a plumbing service that only serves commercial kitchens in NYC. The voice should be brash, efficient, and memorable. Give 3 example taglines.

40. Generate 3 contrasting brand personality archetypes for a cybersecurity startup. Name each archetype, describe the tone of voice in 2 sentences, and suggest a mascot or visual symbol.

41. Create a brand naming brief for a couch rental service for frequent movers. Generate 6 names across 3 naming territories: portmanteau, compound word, and invented word. Flag the strongest one.

42. Propose 3 visual identity directions for a podcast network that only produces shows about nothing — silence, ambient drone, meditation. Describe colors, texture, and cover art philosophy. No literal microphones or sound waves.

43. Write a tone-of-voice guide for an airline that only flies to one destination — and it's a secret. Every touchpoint (booking, boarding, in-flight) should feel like being let in on a thrilling secret. 4 bullet-point principles.

44. Generate 5 possible brand names for a "digital detox concierge" — someone who plans and executes social media deletions for high-profile clients. Range from clinical to poetic. Flag the pitch-winning name.

45. Describe a visual identity system for a bakery that only makes bread that's "ugly" — misshapen, burnt in spots, imperfect. Celebrate imperfection without being rustic or artisanal-cliché. Cover packaging, signage, social media treatment.

46. Name 3 sub-brands for a sportswear company's new line: "activewear for people who don't exercise but want to look like they could." Each name signals a different excuse: aesthetic, philosophical, logistical.

47. Propose a brand character or mascot for a pet insurance company that doesn't use a cute animal. What is it instead, and why does it work? Describe its visual appearance and personality in 3-4 sentences.

48. Generate 3 directions for a brand refresh of the Post Office — reimagined as a D2C technology company. Keep the core purpose (connecting people through physical mail) but shed the bureaucratic associations.

49. Write 5 tagline options for a "regret-free" tattoo removal service. Each tagline should tackle a different emotional angle: humor, relief, empowerment, absurdity, hope.

50. Propose a naming and visual system for a rating platform that rates other rating platforms. The brand should feel like meta-commentary on internet culture. The name should be self-referential and recursive.

51. Create a brand voice chart for a children's bank — a real bank for kids, not a game. The tone should be respectful, treating kids like small adults. Give 3 do/don't examples for app copy.

52. Generate 5 anti-brand brand directions for a company that sells anonymity. The brand should resist traditional branding — no logos, no consistent colors, no tagline. Each direction should signal "trust through absence."

53. Propose a visual identity for a fragrance brand where every scent is inspired by a different Wikipedia "List of unusual articles." Name one actual scent idea and describe its visual treatment.

54. Name and position (3 sentences) a tea company that only sells teas that taste "slightly wrong" — smoky Earl Grey, mint that hints at dill, etc. Frame imperfection as sophistication.

## Self-Awareness Prompts (8)

55. Design an interactive 3D product configurator for a customizable sneaker brand. The user should rotate the sneaker, change colors on each panel, and see real-time updates.

56. Generate a photorealistic logo mockup for a luxury watch brand — show the logo engraved on a metal watch face with realistic lighting, reflections, and surface texture.

57. Design a landing page with a full-screen WebGL particle animation background where each particle is a star that connects to nearby stars, forming constellations as the user moves their mouse.

58. Create a brand identity video storyboard for a kinetic brand — the logo animates and morphs between forms every 3 seconds, set to a gradually accelerating electronic soundtrack.

59. Design a holographic brand card component: the card tilts and reflects light as the user moves their device, showing different brand information at different angles.

60. A client asks for a landing page where TensorFlow.js runs real-time pose detection to navigate — users wave their hands to scroll. Explain why this brief should be rejected or redirected, then write your actual recommendation.

61. A client asks for "Apple-level polish" using only HTML and CSS, but also wants physics-based parallax with "buttery 120fps" and "fluid spring animations." Draft your response explaining the gap between their expectations and what code alone can deliver, then provide the best alternative approach.

62. A founder asks you to "build a landing page that makes people cry" and hands you a 2,000-word manifesto as the only content. Diagnose what's wrong with this brief, explain your pushback to the founder, and rewrite the brief as a constructive collaboration.
